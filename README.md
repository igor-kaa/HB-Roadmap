# Hyperborea Roadmap Scheduler

Локальное web-приложение для построения roadmap игрового проекта Hyperborea.
Загружает CSV с оценками фич, рассчитывает расписание по двухнедельным
спринтам и отображает интерактивную диаграмму Ганта и загрузку команд.

## Запуск

Backend и установка зависимостей не требуются. Откройте `index.html`
непосредственно в браузере.

Исходный набор данных встроен для запуска через `file://`. Другой CSV можно
загрузить кнопкой «Загрузить новый CSV».

Формат входного файла:

```csv
ID,Feature name,Priority,August Build,Dev,GD,TD,Anim
```

`August Build` в текущей версии игнорируется.

## Тесты

Требуется Node.js:

```powershell
npm.cmd test
```

Тесты проверяют regression baseline, зависимости этапов, capacity,
ограничение одного человека на этап фичи и локальный запуск без backend.
